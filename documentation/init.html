Initialisation

In order for SuperScript.Javascript's internal properties to be initialised for each HTTP request, an additional step must be taken: for each HTTP request the default declarations should be added.

The default declarations are those which are declared in the config file or in the SuperScriptConfig class.

Here is an example showing how these default declarations could be added. 

  var defaultDecs = Configuration.Settings.Instance.Declarations;
  foreach (var declaration in defaultDecs)
  {
	  SuperScript.Declarations.AddDeclaration<Declarables.DeclarationBase>(declaration);
  }

The above code could be included in the global.asax file's Application_BeginRequest method.

However, an HTTP handler with the above code is already included inside SuperScript.JavaScript. To enlist this HTTP handler the following code should be added - ideally to the SuperScriptConfig class's RegisterDefaults method. 

  HttpApplication.RegisterModule(typeof (SuperScript.JavaScript.HttpInitialiser));
