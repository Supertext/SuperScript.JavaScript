Comment Declaration

Offers a simple means of declaring a JavaScript comment from server-side code.

The comment will be wrapped in JavaScript multi-line symbols, i.e., /*…*/ 

	class CommentDeclaration : SuperScript.JavaScript.Declarables.DeclarationBase
	{
		override string ToString();
	}
Name Type Description
ToString string

Returns this instance as a formatted JavaScript declaration.

Inherited members
Comment string

This text may be emitted onto its own line or after a variable, depending upon the specific implementation of DeclarationBase.

Optional.

EmitterKey string

Indicates which instance of IEmitter the content should be added to.

Optional. If not specified then the contents will be added to the default implementation of IEmitter.

Name string

Gets or sets the name assigned to this declarable.

Optional. If not specified, SuperScript.JavaScript will use the name of the enumeration.

Example

Comments may be declared in the config file.

Note: The examples on this page assume an instance of IEmitter has been configured with IEmitter.Key = "javascript". This is not configured by default. This page explains how to create and configure an IEmitter. 

	<configuration>
		<configSections>
			<section name="superScript" type="SuperScript.Configuration.SuperScriptConfig, SuperScript.Common" />
			<section name="superScript.JavaScript" type="SuperScript.JavaScript.Configuration.JavaScriptConfig, SuperScript.JavaScript" />
		</configSections>
	</configuration>
	

	…
	

	<superScript.JavaScript>
		<declarations>
			<comment text="This is a test comment." emitterKey="javascript" />
		</declarations>
	</superScript.JavaScript>

Comments may also be declared in code. SuperScript.JavaScript exposes a fluent syntax for declarables which allows for easier adding of declarations. 

	using SuperScript.JavaScript;
	

	Declarations.AddComment(opt => opt.Comment("This is a test comment.")
                      			.EmitterKey("javascript"));

Alternatively, function calls may also be declared in a more conventional programmatic style. 

	using SuperScript.JavaScript;
	

	var cmtDecl = new CommentDeclaration();
	cmtDecl.Comment = "This is a test comment.";
	callDecl.EmitterKey = "javascript";
	SuperScript.Declarations.AddDeclaration<CommentDeclaration>(cmtDecl);

All of the above examples will output the following: 

	/* This is a test comment. */
