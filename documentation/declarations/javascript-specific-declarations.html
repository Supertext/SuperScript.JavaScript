JavaScript-specific Declarations

SuperScript.JavaScript contains a class, DeclarationBase, which is an implementation of the abstract DeclarationBase found in the core SuperScript library. 

			abstract class DeclarationBase : SuperScript.Declarables.DeclarationBase
			{
				string Comment { get; set; }
			}

Aside from some internal logic, DeclarationBase only exposes a single property, Comment, which is common to all implementations of DeclarationBase in the SuperScript.JavaScript library.

Name Type Description
Comment string

This text may be emitted onto its own line or after a variable, depending upon the specific implementation of DeclarationBase.

Implementations
