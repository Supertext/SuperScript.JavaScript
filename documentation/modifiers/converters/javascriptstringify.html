JavaScriptStringify Collection Converter

Converts the Declarations property of the specified PreModifierArgs into an instance of PostModifierArgs where the Emitted property contains the stringified JavaScript.

Inherited members
EmitMode EmitMode

Determines the mode in which this implementation emits contents.

Optional. Default value: EmitMode.Always

This implementation of CollectionConverter simply enumerates each instance of DeclarationBase in the PreModifierArgs.Declarations property and calls DeclarationBase.ToString().

The result of each DeclarationBase.ToString() call is placed on a new line and indented by a tab space.

The result of this conversion process is then passed to the Emitted property of a PostModifierArgs object.
