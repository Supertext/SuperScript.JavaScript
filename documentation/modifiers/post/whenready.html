SuperScript offers a means of declaring assets in one part of a .NET web solution and have them emitted somewhere else.

WhenReady Post-conversion Modifier

Wraps the specified string (in the PostModifierArgs.Emitted property) in a jQuery .ready() function call. 

			class WhenReady : SuperScript.Modifiers.Post.CollectionPostModifier
			{
				EmitMode EmitMode { get; set; }
				string WhenReadyClose { get; set; }
				string WhenReadyOpen { get; set; }
				override PostModifierArgs Process(PostModifierArgs args)
			}
Name Type Description
WhenReadyClose string

Gets or sets the JavaScript string which closes the wrapping function call.

Optional. Default value: "});"

WhenReadyOpen string

Gets or sets the JavaScript string which opens the wrapping function call.

Optional. Default value: "$(function() {"

Process IHtmlString

Wraps the contents in a jQuery DOM-ready closure.

Inherited members
EmitMode EmitMode

Determines the mode in which this implementation emits contents.

Optional. Default value: EmitMode.Always

Like all implementations of CollectionPostModifier, WhenReady exposes a public method, Process which accepts an instance of PostModifierArgs. This argument object's Emitted property contains the output of either

  • an implementation of CollectionPostModifier which has been configured for the context IEmitter, or

  • an implementation of CollectionConverter.

The result of the Process method is an instance of PostModifierArgs where the Emitted property contains the passed-in Emitted property's contents wrapped in a function call as described by the WhenReadyOpen and WhenReadyClose properties.

Important: It should be remembered that multiple implementations of CollectionPostModifier may be configured for an IEmitter.
