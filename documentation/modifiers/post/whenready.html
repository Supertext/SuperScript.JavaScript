WhenReady Post-conversion Modifier

Wraps the specified string (in the PostModifierArgs.Emitted property) in a jQuery .ready() function call.

Like all implementations of CollectionPostModifier, WhenReady exposes a single public method, Process which accepts an instance of PostModifierArgs. This argument object's Emitted property contains the output of either

  • an implementation of CollectionPostModifier which has been configured for the context IEmitter, or

  • an implementation of CollectionConverter.

The result of the Process method is an instance of PostModifierArgs where the Emitted property contains the passed-in Emitted property's contents wrapped in a jQuery .ready() function call.

Important: It should be remembered that multiple implementations of CollectionPostModifier may be configured for an IEmitter.
