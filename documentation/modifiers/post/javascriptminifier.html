JavaScriptMinify Post-conversion Modifier

Minifies the specified string (in the PostModifierArgs.Emitted property) using Microsoft's WebGrease library.

If errors occur during the minification process then an error message is rendered in place.

Like all implementations of CollectionPostModifier, JavaScriptMinifier exposes a single public method, Process which accepts an instance of PostModifierArgs. This argument object's Emitted property contains the output of either

  • an implementation of CollectionPostModifier which has been configured for the context IEmitter, or

  • an implementation of CollectionConverter.

The result of the Process method is an instance of PostModifierArgs where the Emitted property contains the output of the minification process.

Important: It should be remembered that multiple implementations of CollectionPostModifier may be configured for an IEmitter.
