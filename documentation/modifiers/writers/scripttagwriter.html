ScriptTagWriter HTML Writer

Writes the formatted output of the Emitted into an HTML tag.

By default this will output the contents in a <script> tag. 

	class ScriptTagWriter : SuperScript.Modifiers.Writers.HtmlWriter
	{
		EmitMode EmitMode { get; set; }
		override IHtmlString Process(PostModifierArgs args)
		IEnumerable<KeyValuePair<string, string>> TagAttributes { get; set; }
		string TagName { get; set; }
	}
Name Type Description
Process IHtmlString

Executes this instance of HtmlWriter upon the specified argument object.

TagAttributes IEnumerable<KeyValuePair<string, string>>

Gets or sets a collection of key-value pairs which form the tag's attribute collection.

Optional. Default value: a single key-value pair, type="text/javascript".

TagName string

Gets or sets the name of the enclosing HTML tag.

Optional. Default value: "script".

Inherited members
EmitMode EmitMode

Determines the mode in which this implementation emits contents.

Optional. Default value: EmitMode.Always

Like all implementations of HtmlWriter, ScriptTagWriter exposes a single public method, Process which accepts an instance of PostModifierArgs. This argument object's Emitted property contains the output of

  • an implementation of CollectionPostModifier which has been configured for the context IEmitter, or

  • an implementation of CollectionConverter.

The result of the Process method is a string which, in this case, is contained inside an HTML tag described by the TagAttributes and TagName properties.

Example

Programatically adding an instance of ScriptTagWriter to an IEmitter is fairly trivial. However, here is an example of declaratively configuring an IEmitter with a ScriptTagWriter in a config file.

This example also shows how to explicitly set the TagName property. Unfortunately the TagAttributes property cannot be declared in a config file. 

	<configuration>
		<configSections>
			<section name="superScript" type="SuperScript.Configuration.SuperScriptConfig, SuperScript.Common" />
			<section name="superScript.JavaScript" type="SuperScript.JavaScript.Configuration.JavaScriptConfig, SuperScript.JavaScript" />
		</configSections>
	</configuration>
	

	…
	

	<superScript>
		<emitters>
			<emitter key="javascript">
				<writers>
					<modifier type="SuperScript.JavaScript.Modifiers.Writers.ScriptTagWriter, SuperScript.JavaScript">
						<properties>
							<!-- the 'type' attribute has the default value of String, but is shown here by example -->
							<property name="TagName" value="script" type="System.String, mscorlib" />
						</properties>
					</modifier>
				</writers>
			</emitter>
		</emitters>
	</superScript>
